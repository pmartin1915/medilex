# .claude Directory

This directory contains context and documentation for Claude AI sessions working on this project.

## Files

### PROJECT_CONTEXT.md
**The most important file!** Comprehensive overview of:
- Project architecture and file structure
- User's environment and preferences
- Common issues and solutions
- Known bugs and areas for improvement
- Development workflow and commands
- Design philosophy and coding principles

**Always read this first in a new session.**

### NEXT_SESSION_PROMPT.md
Templates for starting new Claude sessions efficiently. Copy/paste these prompts to quickly get Claude up to speed.

### Maintenance

These files should be updated when:
- Significant features are added
- Architecture changes
- New common issues are discovered
- Development workflow changes
- Dependencies are updated

Keep these docs current so future sessions remain productive!
